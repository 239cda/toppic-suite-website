Msconvert - Tutorial

Msconvert is a software tool in ProteoWizard that converts raw files into various formats.

Required Softwares to install MsConvert:

  • MSVC 2008 SP1 (x86)
  • .NET Framework 3.5 SP1
  • .NET Framework 4.0 (Web Installer)

How to Download Msconvert:

Follow the below steps to download it from the website:

  1. Create a new folder named 'Proteowizard' in the C:/ drive of your system in order to save all the extracted files of ProteoWizard.
  2. Go to the link ProteoWizard.
  3. Go to 'Download' option.
  4. From the Download Type > drop down options, choose 'Windows 64-bit tar.bz2' from End Users list.
  5. Agree to the Terms and Conditions.

  6. This downloads 'msconvert.exe' along with all other tools of ProteoWizard.
  7. Save the zipped folder in 'ProteoWizard folder' created earlier.

  8. Extract all the files of the downloaded zipped folder onto your computer.

How to Use Msconvert:

  1. You can download a sample RAW file from here.

    1. The RAW file 'ST_sample.RAW' for Salmonella typhimurium will be downloaded.
    2. Save the file in a convenient location.

  2. To generate the mzXML file from the Input RAW file using msconvert, you can mention the file locations and 'msconvert.exe' location in the command prompt, as shown below.
    1. For Input,as a test,you can use the downloaded sample RAW file.
    2. When you run the below command on command prompt, an output file with .mzXML extension will be created in the location as given in the command prompt for output.

    Sample folder msconvert.exe location:
    C:\ProteoWizard\msconvert.exe
    Sample RAW Input file location:
    C:\TopPIC-Sample\Data\ST_sample.raw
    Sample RAW Output file location:
    C:\TopPIC-Sample\Results\ST_sample.mzXML
    Sample RAW File Command:
    C:\Proteowizard\msconvert.exe --filter "peakPicking true 1-" --mzXML --zlib C:\TopPIC-Sample\Data\ST_sample.RAW -o C:\TopPIC-Sample\Results

    In the above command, the argument "peakPicking true 1-" is used to generate a centroid, not profile, mzXML file, which is needed by the spectral deconvolution tool MS-Deconv. The "-o" argument is to generate output file in the location specified. The size of the input raw file is 689 MB.
