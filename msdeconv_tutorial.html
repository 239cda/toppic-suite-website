Ms-Deconv - Tutorial

MS-Deconv is a top-down spectral deconvolution tool that converts a tandem mass spectrum containing highly charged fragment ions into a list of neutral monoisotopic masses.

How to Download Ms-Deconv:

Click here to download the linux version of MS-Deconv software jar file or follow the below steps to download it manually.

  1. Click on Deconv link here.
  2. Download 'MS-Deconv 0.8.0.7370' by submitting the form.
  3. Both, Windows or Linux versions can be downloaded as a jar file.
  4. Save the file in a convenient location on your system.

  5. This downloads 'MsDeconvConsole-0.8.0.7370.jar' in the desired location.

    6. How to Use Ms-Deconv:

    1. You can download a sample input mzXML file for Salmonella typhimurium (generated from MSconvert) from here.
      1. The sample mzXML file 'ST_sample.mzXML' will be downloaded.
      2. Save the file in a convenient location.
    2. To generate the msalign file from the input mzXML file using MS-Deconv, you can mention the input file location and the 'MsDeconvConsole-0.8.0.7370.jar' location in the command prompt, as shown below. By default, the output file will be generated in the same folder as the input file. To get the output file in a different location from the input file, mention the path in the command after the '-o' parameter.
      1. For Input,as a test,you can use the downloaded sample mzXML file.
      2. When you run the below command on command prompt, an output file with .msalign extension will be created in the location as given in the command prompt for output.

      Sample RAW Input file location:
      C:\TopPIC-Sample\Results\ST_sample.mzXML
      Sample RAW Output file location:
      C:\TopPIC-Sample\Results\ST_sample.mzXML -o msalign
      Sample RAW File Command:
      java -jar C:\TopPIC-Sample\Software\MS-Deconv\MsDeconvConsole-0.8.0.7370.jar C:\TopPIC-Sample\Results\ST_sample.mzXML -o msalign

      In the above command, the argument '-o msalign' is used to generate the file in 'msalign' format. The size of the input mzXML file was 142 MB.
