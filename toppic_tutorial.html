TopPIC - Tutorial

TopPIC (TOP-down mass spectrometry based Proteoform Identification and Characterization) is a software tool for identification and characterization of proteoforms at the whole proteome level by top-down tandem mass spectra using database search.

A computer with a 64-bit Windows 7 operating system is required to run TopPIC.

How to Download TopPIC:

Click on the here to download the TopPIC setup file,"toppic-setup-1.0.0.exe" or follow the below steps to download it manually:

  1. Go to the link TopPIC.
  2. From the 'Installation' part, download the setup file 'toppic-setup-1.0.0.exe'
  3. After completing the registration (optional), accept the license agreement and click download.
  4. This will direct you to the page shown below in the screen shot. Choose the 'Windows Command Line' version.

  5. Save it in a convenient directory on your system.

  6. To install 'toppic-setup-1.0.0.exe' file, double click on the .exe file and follow the screen shots below.

  7. By default, TopPIC software will be installed in the C:\ drive of your system with a 'TopPIC' named folder..

  8. This folder will have the files shown below in the screen shot.

How to Use TopPIC:

  1. You can run TopPIC by using the path to the new default folder created in your C:\ drive after installing TopPIC or you can also copy all the seven files (shown in the above screen shot) to your TopPIC folder where you downloaded the 'topic-setup-1.0.0 exe' file, as shown below. The path to the 'topic.exe' file in the command would change accordingly.

  2. To use TopPIC, you need the following input files:

    1. Fasta file. You can download it directly from here , named 'ST.fasta' or follow the below steps to download it manually.

      • From the Uniprot Database website Uniprot Database. Enter the organism name in the search box.
      • Select reviewed and in the download option choose FASTA (canonical and isoform).
      • Download the fasta file in the desired directory on your computer.The screen shot below shows the reference fasta file for Salmonella typhimurium .

    2. Msalign file from the MS-Deconv output. Sample msalign file can be downloaded from link here.
    3. Modification files (optional)

  3. The final output of TopPIC is OUTPUT table and html file. To generate these, mention the path of msalign file (output of MS-Deconv) and of reference fasta file(from Uniprot Database) as shown below:
    1. For Input, as a test, you can use the downloaded msalign file and reference file(downloaded from sample folder or Uniprot Database)
    2. When you run the below command on command prompt, output files(OUTPUT table and html file) will be created in the location where your input msalign file is located.

    Sample folder toppic-setup-1.0.0.exe location:
    C:\TopPIC \toppic.exe
    Sample input msalign file location:
    C:\TopPIC-Sample\Results\ST_sample_msdeconv.msalign
    Sample input Fasta file location:
    C:\TopPIC-Sample\Data\ST.fasta
    Sample Output file location:
    C:\TopPIC-Sample\Results\ST_sample_msdeconv_html
    C:\TopPIC-Sample\Results\ST_sample_msdeconv.OUTPUT_TABLE
    Sample command:
    C:\TopPIC\toppic.exe C:\TopPIC-Sample\Data\ST.fasta C:\TopPIC-Sample\Results\ST_sample_msdeconv.msalign

    The above command, gives fasta file and msalign file as input. The size of the input msalign file is 6.68 MB and of Salmonella typhimurium reference fasta file is 16.8 KB.The screenshot below shows the output file being processed.
