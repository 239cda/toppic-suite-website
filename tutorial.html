1. Overview

In this tutorial, we use TopPIC to analyze a top-down MS/MS data set of Escherichia coli on a computer with a Windows operating system.

2. Folders

Create the folders below for saving software packages and data sets used in this tutorial.

  1. Create a new folder named 'tutorial' on the C: drive of your system.
  2. Create a new subfolder named 'pw' in the folder C:\tutorial\ for saving the software package Proteowizard.
  3. Create a new subfolder named 'toppic' in the folder C:\tutorial\ for saving the software TopPIC suite.
  4. Create a new subfolder named 'data' in the folder C:\tutorial\ for saving data files.
  5. Create a new subfolder named 'result' in the folder C:\tutorial\ for saving data files.

3. Date set

3.1 Top-down MS/MS date set

The top-down MS/MS data set was generated from protein extract of Escherichia coli K-12 MG1655. The protein extract was analyzed by an RPLC system coupled with an LTQ Orbitrap Velos mass spectrometry. MS1 and MS/MS spectra were collected at a 60,000 resolution with the alternating fragmentation mode. The data set contains 2,027 collision-induced dissociation and 2,027 electron-transfer dissociation top-down MS/MS spectra (see ref. [1]).

Click here to download the data set and save it in the folder C:\tutorial\data\.

3.2 Protein sequence database

The Escherichia coli K-12 MG1655 proteome database was downloaded from UniProt on September 14, 2017. It contains 4,309 protein sequences.

4. Software tools

4.1 Msconvert

Msconvert is a software tool in ProteoWizard that converts raw files into various formats.

4.1.1 Required software packages to install ProteoWizard

  • MSVC 2008 SP1 (x86)
  • .NET Framework 3.5 SP1
  • .NET Framework 4.0 (Web Installer)

4.1.2 Download ProteoWizard

Follow the steps to below download ProteoWizard:

  1. Go to the link ProteoWizard.
  2. Choose the download type 'Windows 64-bit tar.bz2' for end users.

  3. Click the button "I agree to the licensing terms, download ProteoWizard" to download ProteoWizard and save the bz2 file in the folder C:\tutorial\pw\.

  4. Extract all the files of the downloaded bz2 file to the folder C:\tutorial\pw\.


4.2 TopPIC suite

4.2.1 Download TopPIC suite

  1. Go to the link to download webpage of TopPIC.
  2. Choose the download type "Windows 64-bit zip file." Fill out the registration form. Click "I agree to the license agreement and download" to download TopPIC suite.


  3. Save it to the folder C:\tutorial\toppic\.

  4. Extract all the files of the downloaded zip file to the folder C:\tutorial\toppic\.

4 Data analysis

4.1 File format conversion

We use msconvert to convert the raw file ecoli.raw to an mzML file ecoli.mzML.

To generate the mzXML file from the Input RAW file using msconvert, you can mention the file locations and 'msconvert.exe' location in the command prompt, as shown below.
  1. For Input,as a test,you can use the downloaded sample RAW file.
  2. When you run the below command on command prompt, an output file with .mzXML extension will be created in the location as given in the command prompt for output.

Sample folder msconvert.exe location:
C:\ProteoWizard\msconvert.exe
Sample RAW Input file location:
C:\TopPIC-Sample\Data\ST_sample.raw
Sample RAW Output file location:
C:\TopPIC-Sample\Results\ST_sample.mzXML
Sample RAW File Command:
C:\Proteowizard\msconvert.exe --filter "peakPicking true 1-" --mzXML --zlib C:\TopPIC-Sample\Data\ST_sample.RAW -o C:\TopPIC-Sample\Results

In the above command, the argument "peakPicking true 1-" is used to generate a centroid, not profile, mzXML file, which is needed by the spectral deconvolution tool MS-Deconv. The "-o" argument is to generate output file in the location specified. The size of the input raw file is 689 MB.
